## ðŸ“± Screen Time Setup (iOS - iPhone & iPad)

**Recommended for**: Families using Apple devices

### Step-by-Step Setup (Parent Device)

#### 1. Enable Screen Time

1. Open **Settings**
2. Tap **Screen Time**
3. Tap **Turn On Screen Time**
4. Select **This is My Child's iPhone/iPad**

#### 2. Set Downtime

1. In Screen Time, tap **Downtime**
2. Toggle **Downtime** ON
3. Set schedule:
   - **School Nights**: 8:00 PM - 7:00 AM
   - **Weekends**: 9:00 PM - 7:00 AM
4. Customize based on your family

#### 3. Configure App Limits

1. Tap **App Limits**
2. Tap **Add Limit**
3. Select **All Apps & Categories** or specific categories:
   - **Education**: 2-3 hours per day
   - **Entertainment**: 1 hour per day
   - **Social**: 30 min per day (if applicable)
4. Tap **Add** and confirm

**Recommended Limits for AI Learning:**
- Ages 7-8: 45-60 min/day
- Ages 9-10: 60-90 min/day
- Ages 11-12: 90-120 min/day

#### 4. Content & Privacy Restrictions

1. Tap **Content & Privacy Restrictions**
2. Toggle **Content & Privacy Restrictions** ON
3. Configure:

**iTunes & App Store Purchases:**
- Installing Apps: **Don't Allow** or **Ask**
- Deleting Apps: **Don't Allow**
- In-App Purchases: **Don't Allow**

**Allowed Apps:**
- Ensure Safari, Chrome (if needed), and educational apps are enabled

**Content Restrictions:**
- **Web Content**: Select **Limit Adult Websites**
- Add specific websites to **ALLOWED** list:
  - chat.openai.com (ChatGPT)
  - gemini.google.com (Google Gemini)
  - bing.com/chat (Copilot)
  - poe.com (Poe)
  - scratch.mit.edu (Scratch)
  - khanacademy.org (Khan Academy)

**Privacy Settings:**
- Location Services: Review per app
- Photos: **Ask** for each app
- Microphone: **Ask** for each app
- Camera: **Ask** for each app

#### 5. Set Screen Time Passcode

1. Tap **Use Screen Time Passcode**
2. Create a passcode **different from device passcode**
3. Store it safelyâ€”you'll need it to adjust settings

### Monitoring on iOS

**View Reports:**
1. Open **Screen Time**
2. Tap **See All Activity**
3. Review:
   - Daily/weekly usage
   - Most used apps
   - Pickups and notifications
   - Website visits

**Weekly Review Routine:**
- Every Sunday evening, review the week's report
- Discuss with kids what they learned
- Adjust limits if needed

## ðŸ¤– Family Link Setup (Android & Chromebook)

**Recommended for**: Families using Android phones, tablets, or Chromebooks

### Step-by-Step Setup

#### 1. Install Family Link

**On Parent Device:**
1. Download **Google Family Link (for parents)** from Play Store
2. Open app and tap **Get Started**
3. Select your Google account
4. Tap **Yes** to be a family manager

**On Child Device:**
1. Download **Google Family Link (for children & teens)**
2. OR use existing device's **Settings > Google > Parental Controls**

#### 2. Create Child Account

1. In Family Link parent app, tap **+** to add child
2. Choose:
   - **Create account** if child doesn't have Google account
   - **Add existing account** if they do
3. Follow prompts to enter:
   - Child's name
   - Date of birth
   - Create username (firstname.lastname@gmail.com)
   - Create password (help them choose a strong one)
4. Agree to parental consent
5. Link child account to child device

#### 3. Configure App Permissions

1. In Family Link app, select child's name
2. Tap **Controls**
3. Go to **Google Play**:
   - Set **All content**: Require approval for all downloads
   - Review purchases: **Always require password**

#### 4. Set Screen Time Limits

1. In **Controls**, tap **Daily limit**
2. Set limits per day:
   - **Monday-Friday**: 1-2 hours
   - **Saturday-Sunday**: 2-3 hours
3. Adjust based on age and needs

**Set Bedtime:**
1. Tap **Bedtime**
2. Set device sleep schedule:
   - School nights: 8:00 PM - 7:00 AM
   - Weekends: 9:00 PM - 7:00 AM

#### 5. Manage Apps

1. Go to **App Activity**
2. Review installed apps
3. Block or allow specific apps
4. Set time limits per app

**Approve Safe AI Tools:**
- Chrome browser (for ChatGPT, Gemini, Bing)
- Google Gemini app (if available)
- Scratch Jr or Scratch app
- Khan Academy app

#### 6. Web Filtering

1. In **Controls**, tap **Filters on Google Chrome**
2. Select:
   - **Try to block mature sites** (recommended)
   - OR **Only allow certain sites** (most restrictive)
3. Manage sites:
   - Tap **Approved** to add safe AI sites:
     - chat.openai.com
     - gemini.google.com
     - bing.com/chat
     - poe.com
     - scratch.mit.edu
     - khanacademy.org

#### 7. Location Tracking

1. In **Controls**, tap **Location**
2. Toggle **ON** if you want to see device location
3. Useful for Chromebooks/tablets that leave home

### Monitoring on Family Link

**View Activity:**
1. Open Family Link parent app
2. Select child's name
3. Review:
   - Screen time today
   - Apps used
   - Websites visited
   - Activity highlights

**Weekly Reports:**
- Emailed to parent every week
- Shows trends and patterns

## ðŸŒ Network-Level Protection (Router & DNS)

**Recommended for**: All families, adds extra protection layer

### Option 1: Router Parental Controls

**Most modern routers have built-in controls:**

1. Log into your router (check manual for IP address, usually 192.168.1.1)
2. Find **Parental Controls** section
3. Enable controls and:
   - Add children's devices by MAC address
   - Set time schedules for internet access
   - Block categories: Adult, Gambling, Violence
   - Allow categories: Education, Reference

**Popular Router Brands:**
- **Google WiFi/Nest WiFi**: Use Family WiFi feature
- **Eero**: Use Eero Secure
- **Netgear**: Has built-in Circle parental controls
- **TP-Link**: HomeCare parental controls

### Option 2: FireWalla (Recommended by Issue Author)

**Current Setup Note**: Family already has FireWalla in monitoring modeâ€”needs to be refined for active filtering.

**FireWalla Setup:**
1. Open FireWalla app
2. Go to **Family Protect**
3. Create profiles for each child
4. Assign devices to profiles
5. Configure:
   - **Safe Search**: Force on Google, Bing, YouTube
   - **Content Filtering**: Enable age-appropriate filters
   - **Time Restrictions**: Set internet access hours
   - **Monitor Mode â†’ Active Mode**: Switch to enforce, not just monitor

**FireWalla Monitoring:**
- Real-time alerts for blocked content
- Daily/weekly reports
- See what kids are accessing

### Option 3: DNS Filtering (Free)

**CleanBrowsing DNS** (Free family-safe DNS):
1. Open router settings
2. Find **DNS Settings**
3. Replace DNS servers with:
   - Primary: `185.228.168.168`

4. Save and restart router

**OpenDNS FamilyShield** (Free):
- Primary: `208.67.222.123`

**What this does**: Blocks adult content at DNS level for ALL devices on network.

## â“ FAQ

**Q: Won't my kids think I don't trust them?**
A: Frame it as **safety**, not surveillance. "We have rules and checks to keep you safe online, just like seatbelts in the car."

**Q: What if my kids find ways around the controls?**
A: Have a conversation, not a punishment. Explain WHY controls exist. If they're curious and problem-solving, channel it positively!

**Q: How do I balance safety and independence?**
A: Increase independence gradually. Ages 7-8 = high supervision. Ages 11-12 = more trust but still monitoring. By 13+, focus on open communication.

**Q: My parental controls keep drifting out of compliance. Help!**
A: Set monthly calendar reminder to verify settings. iOS Screen Time passcode helps prevent changes. Family Link can be managed remotely.

**Q: Should I monitor everything they do?**
A: Balance! Weekly spot checks > constant surveillance. Build trust while maintaining safety.

**Q: What if I make a mistake with the settings?**
A: That's okay! This is learning for parents too. Adjust as you go.
