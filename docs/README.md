Welcome to the AI-For-Everyone documentation!

## Available Documentation

### [AI for Kids & Families](./kids/)
A comprehensive guide for teaching children (ages 7-12) how to use AI safely and creatively at home. Perfect for parents, homeschool families, and after-school programs.

**Key resources:**
- [Computer Lab Setup Guide](./kids/computer-lab-guide.md) - Hardware recommendations for any budget
- [Safety Setup](./kids/safety-setup.md) - Child safety and parental controls
- [Kids AI Curriculum v0.1](./kids/kids-ai-curriculum-v0.1.md) - Complete 18-24 week curriculum
- [Kid Projects](./kids/kid-projects.md) - 14 ready-to-start projects
- [Parent Handbook](./kids/parent-handbook.md) - Teaching guide for non-technical parents
- [GitHub Projects Template](./kids/github-project-template.md) - Family learning board setup

### [AI Governance](./ai-governance.md)
Learn about how we use AI tools in this project, how to attribute AI assistance, and why transparency matters.

**Key topics:**
- AI label taxonomy (`ai:chatgpt`, `ai:copilot-editor`, etc.)
- How to attribute AI in issues and PRs
- Automated labeling and template usage
- Best practices for AI-assisted contributions

## Scripts

### [create-labels.sh](./scripts/create-labels.sh)
Helper script to create all the AI governance labels in the repository. Requires GitHub CLI (`gh`) and appropriate permissions.

**Usage:**
```bash
cd /path/to/AI-For-Everyone
./docs/scripts/create-labels.sh
```

## For Contributors

When contributing to AI-For-Everyone:

1. **Use our templates** when creating issues or PRs
2. **Check the AI Assistance boxes** to indicate which AI tools you used
3. **Be transparent** about AI usage - it helps everyone learn!
4. **Review [ai-governance.md](./ai-governance.md)** for detailed guidance

## For Learners

This repository is itself an educational resource! You can:

- **Learn from examples** of AI-assisted development

- **Follow patterns** that work in real projects
