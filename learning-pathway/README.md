### Follow the Pathway

For a structured learning experience, follow this order:

1. **[AI Basics](./modules/basics.md)** - Understand what AI is (30-45 minutes)
2. **[Safety & Guidelines](./modules/safety.md)** - Learn to use AI safely (20-30 minutes)
3. **[Free AI Tools](./modules/tools.md)** - Discover which tools to use (15-20 minutes)
4. **[Teaching Kids About AI](./modules/teaching-kids.md)** - Help children learn (45-60 minutes)

Total time: ~2-3 hours to complete the full pathway

### Learn as a Family

**Family Learning Sessions** (recommended approach):
- Set aside 30-45 minutes each week
- Work through one module together
- Try the activities as a family
- Discuss what you learned over dinner
- Practice between sessions

**Tips for Family Learning:**
- Let kids ask questions and explore
- Parents and kids learn together‚Äîno one needs to be an expert
- Make it fun with creative projects
- Celebrate small wins and discoveries
- Take breaks when needed

### Build Your Skills

- Use AI tools confidently in daily life

- Make informed decisions about AI tools and services
- Teach others about AI basics
