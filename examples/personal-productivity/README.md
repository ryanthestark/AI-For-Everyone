## üöÄ Quick Start Examples

### 1. Set Up Weekly Planning System (1 hour setup, 30 min/week)

**Scenario:** You want better organization and less overwhelm.

**Tools:** ChatGPT + GitHub Projects or paper planner

**Process:**

**Setup (One-time):**
1. List all your regular commitments
2. Identify time blocks available
3. Choose planning time (Sunday evening?)
4. Create planning template
5. Set up tracking system

**Weekly Routine:**
1. Review last week (10 min)
2. Dump all to-dos (5 min)
3. Prioritize and schedule (10 min)
4. Prepare for week (5 min)

**Sample Prompt:**
```
Help me create a weekly planning system.

My situation:
- Full-time job (9-5)
- 2 kids (school pickup at 3:30)
- Want: exercise 3x/week, side project 5 hours/week
- Evening activities: dinner, homework, family time

Create a template for Sunday planning that:
1. Reviews the past week
2. Plans the upcoming week
3. Prioritizes tasks
4. Schedules specific work blocks
5. Keeps it realistic (30 min max)
```

### 2. Build a Morning Routine (2 weeks to establish)

**Scenario:** Mornings are chaotic; want calm, productive starts.

**Tools:** ChatGPT + habit tracker

**Process:**
1. **Week 1:** Design ideal routine, start with 1-2 habits
2. **Week 2:** Add more elements, refine timing
3. **Week 3+:** Maintain and adjust

**Sample Prompt:**
```
Help me design a morning routine.

Current situation:
- Wake up: 6:30 AM
- Leave house: 8:00 AM (90 minutes available)
- Goals: exercise, healthy breakfast, organized start

Challenges:
- Hit snooze often
- Rush out the door
- Skip breakfast

Create:
1. A realistic 90-minute routine
2. Tips to wake up easier
3. How to build this gradually
4. Troubleshooting common issues
```

### 3. Prioritize Daily Tasks (15 min daily)

**Scenario:** Long to-do list, don't know where to start.

**Tools:** ChatGPT or paper

**Process (Daily):**
1. Brain dump all tasks (5 min)
2. Categorize by urgency/importance (5 min)
3. Choose top 3 for today (2 min)
4. Schedule them (3 min)

**Sample Prompt:**
```
Help me prioritize today's tasks using the Eisenhower Matrix.

My tasks:
- Finish project proposal (due Friday)
- Respond to 15 emails
- Grocery shopping
- Call about insurance
- Exercise
- Plan weekend trip
- Read for book club
- Clean garage
- Learn new software

Context:
- Today is Tuesday
- I have 4 hours of focused work time
- Meeting at 2 PM

Help me:
1. Categorize these tasks
2. Choose top 3 for today
3. Suggest when to do others
4. Identify what can be deleted/delegated
```
