## üìñ Template Anatomy

Good prompts include:

**1. Role/Context**
```
You are a helpful tutor explaining concepts to a [grade level] student.
```

**2. Task**
```
Please explain [concept] in simple terms.
```

**3. Requirements**
```
- Use analogies and examples
- Avoid technical jargon
- Check my understanding with questions
```

**4. Format**
```
Structure your explanation as:
1. Simple definition
2. Real-world example
3. Practice question
```

## ‚úÖ Prompt Template Best Practices

**Do:**
* ‚úÖ Be specific about what you want
* ‚úÖ Provide relevant context
* ‚úÖ Specify the format you need
* ‚úÖ Include your level/background
* ‚úÖ Ask for verification or examples

**Don't:**
* ‚ùå Be too vague ("help me with math")
* ‚ùå Forget to specify your level
* ‚ùå Ask for too many things at once
* ‚ùå Assume AI knows your context
* ‚ùå Accept first response without iteration

## üí° Example: Before and After

**Before (Vague):**
```
Tell me about photosynthesis
```

**After (Using Template):**
```
You are a biology tutor explaining concepts to a 7th-grade student.

Please explain photosynthesis in simple terms.

Requirements:
- Use a real-world analogy
- Explain the inputs and outputs
- Avoid complex chemistry terms
- Include a simple diagram description

Format:
1. One-sentence summary
2. Analogy to everyday life
3. Step-by-step process
4. Why it matters
5. A question to check my understanding
```

## üåü Tips for Better Prompts

**Be specific:**
* "Explain algebra" ‚Üí "Explain solving for x in linear equations"
* "Help with essay" ‚Üí "Help me outline a 5-paragraph persuasive essay"

**Add context:**
* Include your knowledge level
* Mention any constraints (time, length)
* Specify your goal

**Request format:**
* Bullet points vs. paragraphs
* Step-by-step vs. summary
* With examples vs. theory only

**Iterate:**
* First response not quite right? Ask for adjustments
* "Make it simpler"
* "Add more examples"
* "Focus on [specific aspect]"
