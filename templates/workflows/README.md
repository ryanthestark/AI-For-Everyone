## üìñ Workflow Structure

Each workflow template includes:

### Overview
* **Purpose** - What this workflow accomplishes
* **Time required** - How long it typically takes
* **Tools needed** - What you'll use
* **Prerequisites** - What you need first

### Steps
Each step includes:
* **Step number** - Order matters
* **Action** - What to do
* **Details** - How to do it
* **Tips** - Make it easier
* **Output** - What you should have

* **Common issues** - Troubleshoot problems

## üí° Example Workflow Snippet

**Homework Completion Workflow - Step 3**

### Step 3: Break Down the Assignment

**Action:** Divide the homework into smaller tasks

**Details:**
1. Read the assignment completely
2. Identify distinct parts or questions
3. Estimate time for each part
4. Order tasks by difficulty or dependency

**Tips:**
* Start with what you understand best
* Break big questions into sub-questions
* Use AI to help identify steps if stuck

**Output:** 
A task list with time estimates

**Checkpoint:**
‚úì Each task is clear and specific
‚úì Time estimates total less than available time
‚úì No task seems overwhelming
